# **App Name**: 客盈宝

## Core Features:

- 合作伙伴注册/登录: 合作伙伴注册/登录：合作伙伴可以通过注册账号，等待超级管理员审核通过后登录系统。
- 客户报备: 客户报备：合作伙伴可以录入客户的详细信息，包括客户身份证号码等。
- 客户查重: 客户查重：系统使用AI tool，以客户身份证号码为唯一标识，自动检测客户是否已被其他合作伙伴报备，若重复则禁止报备，但是允许提交申诉。
- 申诉管理: 申诉管理：如果客户信息重复，合作伙伴可以提起申诉，说明情况，等待管理员处理。
- 数据导出: 数据导出：超级管理员可以导出所有合作伙伴用户及客户信息，方便数据分析和管理。

## Style Guidelines:

- 主色调：使用清新的浅蓝色 (#E3F2FD) 作为主色调，营造信任和专业感。
- 辅助色：使用稳重的灰色 (#607D8B) 作为辅助色，用于文本和背景。
- 强调色：使用鲜亮的橙色 (#FF9800) 作为强调色，突出重要操作按钮和提示信息。
- 简洁现代的字体，保证在各种设备上的清晰可读性。
- 采用响应式布局，保证在各种设备上的最佳显示效果。
- 使用简洁的线性图标，增强页面的现代感。

## Original User Request:
帮我开发一个合作伙伴客户报备系统，要求超级管理员，可以管理合作伙伴和合作伙伴提交的所有客户信息，具备合作伙伴用户及所有客户信息的导出功能及管理，要求系统简单、极致的美观、合作伙伴注册需要超级管理员审核通过，报备的客户需要有详细信息，以客户身份证号码作为标识，判断客户是否重复提交，如果重复、不能报备，但是可以提起申诉说明情况，管理员可以及时进行处理。合作伙伴之间所有信息只有自己的账号登录后看自己报备的客户，系统具备基本的Crm功能，全站要求全中文界面。
